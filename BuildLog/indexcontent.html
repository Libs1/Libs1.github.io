Software Controlled Drone - Build Instructions

Introduction

The Software Controlled Drone project will allow users to control a Syma X12 Nano QuadCopter Drone with an Android application, which will use an Arduino and a nRF24L01 transciever to communicate with the drone

This is the pin set up from the Arduino to the nRF24L01 Transceiver:

5v -> VCC
GND -> GND
D8 -> IRQ
D9 -> CE
D10 -> CSN
D11 -> MOSI
D12 -> MISO
D13 -> SCK
