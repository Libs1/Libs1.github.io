Software Controlled Drone - Build Instructions

Introduction

The Software Controlled Drone project will allow users to control a Syma X12 Nano QuadCopter Drone with an Android application, which will use an Arduino and a nRF24L01 transciever to communicate with the drone

Bill of Materials/Budget

Item Quantity Total Price(With Tax + Shipping)
Arduino Uno R3 1 $30.95
SparkFun Transceiver Breakout - nRF24L01 1 $29.37
Syma x12 Nano 6-Axis Gyro 4CH RC Quadcopter Drone 1 $49.37
AA Batteries 1 pack(containing 4 batteries) $5.64

Time Commitment

Task Time To Complete The Task
Ordering Parts 3 Days(Shippings)
Wiring the nRF24L01 transceiver to the Arduino 20 minutes
Downloading the Arduino IDE 10 minutes
Importing the nRF24L01 library to the Arduino 10 minutes
Running the code 5 minutes
